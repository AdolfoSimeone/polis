EXTRAORDINARY COMMUNICATION FOR LARGE GROUPS

Broadcast is a solved problem. We built Polis because communicating back to a broadcasting entity is not.

Polis helps you get a comprehensible, sophisticated response from a scalably large, decentralized group - something thus far impossible in the course of human communication. Polis is a web application that leverages real-time connectivity, advanced statistics, machine learning and interactive data visualization to deliver a productive, engaging experience whether users are interacting in person or spread out across the globe.
